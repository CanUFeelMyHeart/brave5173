# Инструкция по связке Git - GitHub

Чтобы склонировать репозиторий к себе на ПК, используется команда: 
> **git clone URL** - клонируется репозиторий с адресом **URL** на локальный ПК
> **git pull** - получение изменений из удаленного репозитория и слияние с локальной версией
> **git push** - отправляет локальную версию репозитория на внешний(удаленный) репозиторий
